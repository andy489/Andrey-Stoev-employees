Employee Pair Finder

Find employees who worked together the longest

Upload a CSV file with employee assignment data.
CSV Format Requirements:

The CSV file should contain the following columns:

  • EmpID - Employee ID (integer)
  • ProjectID - Project ID (integer)
  • DateFrom - Start date (multiple formats supported)
  • DateTo - End date (NULL for current date)

Example:
143, 12, 2013-11-01, 2014-01-05
218, 10, 2012-05-16, NULL
143, 10, 2009-01-01, 2011-04-27
